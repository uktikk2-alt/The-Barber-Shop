# Premium Barbershop Landing Page

A high-end, responsive landing page for a premium barbershop, featuring modern design trends like glassmorphism, advanced scroll animations, and interactive components.

## ✨ Key Features

- **Global Glassmorphism**: A sleek design language using semi-transparent layers and backdrop blurs for a premium look.
- **Scroll-Triggered Color Flip**: A custom GSAP-powered experience where the features section transitions from white to absolute black upon scrolling.
- **Interactive Experience**:
  - **GSAP Counters**: High-performance number counter animations for the stats section.
  - **Animated Navigation**: A custom-coded hamburger menu that morphs into a close icon on mobile.
  - **Interactive Services**: A hover-triggered accordion that dynamically swaps preview images.
- **Micro-Animations**: 3D icon rotations, magnetic buttons, and hover-triggered glow effects.
- **Optimized for Mobile**: Fully responsive layout with custom spacing logic using CSS `clamp()` for perfect typography on any screen size.
- **Performance First**: Built with vanilla HTML, CSS, and JS. Zero build steps required—just pure web performance.

## 🚀 Tech Stack

- **Core**: HTML5, CSS3, JavaScript (ES6+)
- **Animations**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger), [AOS](https://michalsnik.github.io/aos/) (Animate on Scroll)
- **Icons**: FontAwesome 6
- **Fonts**: Google Fonts (Oswald & Inter)

## 📦 Deployment

This project is tailored for static hosting services like **Vercel**, **GitHub Pages**, or **Netlify**.

### Deploy to Vercel
1. Upload this repository to GitHub.
2. Connect your GitHub account to [Vercel](https://vercel.com).
3. Select this repository and click **Deploy**. Vercel will automatically detect the `index.html` and host it as a static site.
